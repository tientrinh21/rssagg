# RSS Aggregatar

## Tech Used

-   **Languages:** Go
-   **Library:** [google/uuid](https://github.com/google/uuid), [go-chi](https://go-chi.io/), [godotenv](https://github.com/joho/godotenv)
-   **Database:** PostgreSQL
-   **Tools:**
    -   [goose](https://github.com/pressly/goose) - database migration
    -   [sqlc](https://sqlc.dev/) - SQL compiler

## Quick Start

```sh
cd rssagg
go build && ./rssagg
```
